## Project Overview
Built directly from Figma designs, this project focuses on high-fidelity, pixel-perfect implementation.

## Tech Stack
Framework: Next.js (React)
Styling & Animation: Tailwind CSS & Framer Motion
UI Components: shadcn/ui
Deployment: Vercel
Key Features
Responsive Layout: Fully optimized for seamless viewing across all device sizes.
Reusable Components: Developed with a modular architecture for scalability.
Image Optimization: Leveraging Next.js for high performance and fast load times.

## AI Usage
AI tools were utilized to architect the logic for the multi-step form system and suggest minor performance optimizations. The core implementation, styling, and integration of functionalities were authored manually to ensure precision.
